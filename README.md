This package is an API wrapper to interact with [fixer.io](https://fixer.io) service.

## Install

Via Composer

``` bash
$ composer require victoravelar/fixer-exchange
```

## Usage

``` php
$skeleton = new VictorAvelar\Fixer();
echo $skeleton->echoPhrase('Hello, League!');
```
